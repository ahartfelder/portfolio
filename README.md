# Portfólio de Desenvolvimento

Este repositório reúne projetos que demonstram minhas habilidades como desenvolvedor de software, com foco em aplicações web modernas, performáticas e escaláveis.

Os projetos incluem soluções construídas com **Next.js, Node.js, TypeScript, PostgreSQL** e integrações via **APIs RESTful**, sempre aplicando **boas práticas de código, performance e arquitetura**.

## 🚀 Tecnologias e Ferramentas

- **Frontend:** Next.js, React, HTML5, CSS3
- **Backend:** Node.js, Express, Nest.js
- **Banco de Dados:** PostgreSQL, MongoDB
- **Outros:** Redis, APIs RESTful, Git, GitHub
- **Infraestrutura:** AWS S3, Docker

## 📌 Estrutura dos Projetos

Cada projeto contém:

1. **Código-fonte limpo e organizado** seguindo boas práticas.
2. **Documentação** para facilitar o entendimento e uso.
3. **Demonstração** (quando aplicável) para visualização prática do resultado.

## 🛠 Como Executar Localmente

1. **Clone este repositório**

```bash
git clone https://github.com/ahartfelder/portfolio.git
```

2. **Instale as dependências**

```bash
npm install
```

3. **Inicie o servidor de desenvolvimento**

```bash
npm run dev
```

4. Acesse em: http://localhost:3000
